# AI Interview Preparation Platform

An AI-powered interview preparation platform built with React and Vite.

## Features

- AI-generated interview questions
- HTML, CSS, JavaScript, React categories
- Voice-to-text answers
- AI feedback using OpenRouter
- Interview timer
- Progress tracking with Local Storage
- Dark/Light mode
- Responsive design

## Tech Stack

- React.js
- Vite
- JavaScript
- CSS
- OpenRouter API
- Web Speech API
- Local Storage
